## Run Locally

Clone the project

```bash
  git clone https://github.com/juanzitelli/caldar-web
```

Go to the project directory

```bash
  cd caldar-web
```

Install dependencies

```bash
  npm install
```

Start the server on development mode

```bash
  npm run dev
```

Or build the app and run in production mode

```bash
  npm run build
  npm start
```
